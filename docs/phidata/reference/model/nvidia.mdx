## Example

<CodeGroup>

```python agent.py
from phi.agent import Agent
from phi.model.nvidia import Nvidia

agent = Agent(model=Nvidia(id="nvidia/llama-3.1-nemotron-70b-instruct"), markdown=True)

agent.print_response("Share a 2 sentence horror story")
```
