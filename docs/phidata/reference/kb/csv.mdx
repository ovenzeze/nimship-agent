## CSVKnowledgeBase Params

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |

| `reader` | `CSVReader` | `CSVReader()` | A `CSVReader` that reads the CSV file and converts it into `Documents` for the vector database |
